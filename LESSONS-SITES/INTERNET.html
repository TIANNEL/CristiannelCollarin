The Internet

Definitions

  • The Internet: A global system of interconnected computer networks that communicate via the Internet Protocol Suite (TCP/IP).
  • Cloud Computing: The on-demand delivery of IT resources (like compute power and database storage) via the internet with pay-as-you-go pricing.
  • Internet Protocols: Specialized rules for data transfer, including HTTP (web data), FTP (file transfer), and TCP/IP (network connection).

History and Theoretical Concepts

  • Historical Timeline:
    • 1960s-1969: Creation of ARPANET, the precursor to the modern internet.
    • 1980s: The rise of Internet Service Providers (ISPs).
    • 1990s: Invention of the World Wide Web (WWW) by Tim Berners-Lee.
    • 2010s-2020s: Shift toward Cloud Computing and ubiquitous connectivity.
  • Cloud Deployment Models:
    • Public Cloud: Owned and operated by third-party providers (e.g., AWS, Google Cloud).
    • Private Cloud: Exclusively used by a single organization.
    • Hybrid Cloud: Combines both public and private infrastructures.

Uses of the Internet

  • Search Information: Everything from documents to books is searchable using engines like Google, Bing, and Yahoo.
  • Digital News: Real-time, paperless delivery of news that is easily accessible compared to traditional television or print.
  • Electronic Mail (E-mail): A core tool for professional and personal communication and data transfer.
  • Search Engine Marketing: Utilizing the internet for business growth, advertising, and reaching global markets.

Web Security and Best Practices

  • Importance of Security: Protecting sensitive information from hackers to prevent the spread of malware and data theft.
  • Security Indicators:
    • HTTPS: Look for the "S" (Secure) and the padlock icon in the browser address bar.
    • Site Authority: Check if the website belongs to an established, trustworthy institution.
  • Safety Strategies:
    • Use Strong Passwords and Two-Factor Authentication (2FA).
    • Avoid posting private email addresses in public forums.
    • Always use secure, encrypted networks rather than public Wi-Fi.
