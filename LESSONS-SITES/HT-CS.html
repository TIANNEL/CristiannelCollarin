HTML5 & CSS

Definitions

  • HTML (HyperText Markup Language): The "mother tongue" of websites, used to create and link web pages together using tags.
  • CSS (Cascading Style Sheets): A language used to describe the presentation of a document, including colors, fonts, and layouts.
  • Tags: The basic coding units of HTML, enclosed in angle brackets (< >), that instruct the browser on how to display content.
  • Attributes: Special words used inside a tag to provide additional information or control the element's behavior.

Theoretical Foundations

  • Separation of Concerns: The concept where HTML handles the structure of the page, while CSS manages the design and visual aesthetics.
  • The Box Model: A core CSS concept where every element is considered a rectangular box consisting of margins, borders, padding, and the actual content.
  • Selectors and Rules: CSS works by "selecting" an HTML element (like a <h1> or a class) and applying a declaration block (property and value) to it.

Web Document Structure

  • Standard Elements:
    • <!DOCTYPE html>: Defines that the document is an HTML5 file.
    • <html>: The root element of an HTML page.
    • <head>: Contains meta information about the document (like the title or links to CSS).
    • <body>: Contains the visible content of the page.
  • Block-Level Tags: Elements like <div> (a container for sections) and <h1-h6> (headings) that start on a new line.

Styling Methods & Best Practices

  • Three Ways to Apply CSS:
    • Inline: Using the style attribute inside HTML tags.
    • Internal: Using the <style> element in the <head> section.
    • External: Linking to a separate .css file using the <link> tag (the most efficient method).
  • Classes vs. IDs:
    • Classes (.): Can be used to style multiple elements on a page.
    • IDs (#): Used to uniquely identify and style a single specific element.
